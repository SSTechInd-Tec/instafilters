### Instagram Image Filters 

This is a fun project 


### Setting Up

```bash
npm install
```

or 

```bash
yarn install
```



If you want to start it in `dev` mode, use following command:

```bash 
npm run dev
```

or 

```bash
yarn run dev
```

If you want to build it and start it, use following command:

```bash
npm run build
npm run start
```

or 

```bash
yarn run build
yarn run start
```
